Volcano Exploration

Endpoints

/new-game/:difficulty/

Difficulty: easy, medium, hard

/move/:direction

Direction: up, down, left, right

/player/

Returns the health and moves remaining from the player

/map/

Returns the map with the player location

Uses traditional ASCII characters for identification

'@' is the player

'.' is a blank tile

'S' is a speeder tile

'L' is a lava tile

'M' is a mud tile

'H' is the start tile

'X' is the end tile

/world-status/

Returns a JSON with the current world, player, and difficulty.

To be used with a custom frontend for a complete game stats.

Game can be played completely via HTTP endpoints without ever using this.
